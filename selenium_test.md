# Selenium Basic Project

This is a complete beginner-to-intermediate Selenium automation project in Python.

## Features Covered

- WebDriver setup
- Browser navigation
- All major locator strategies
- SendKeys and Click
- Implicit and Explicit Wait
- Screenshot capture
- Cookies handling
- Execute JavaScript
- Dropdown (Select class)
- Alerts handling
- Frames switching
- Multiple Windows/Tabs
- Mouse Hover
- Drag and Drop
- Keyboard actions
- Exception handling

---

## Installation

1. Clone project
2. Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Run Project

```bash
python main.py
```
